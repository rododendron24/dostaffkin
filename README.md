# Dostaffkin

Приложение для заказа доставки посылок между странами и внутри стран. Пользователи могут создавать заказы, отслеживать статус доставки и просматривать информацию о сервисе.

## Функционал

- **Главная** — информация о сервисе доставки
- **Заказать** — создание нового заказа доставки
- **Отследить** — отслеживание статуса посылки по ID

## Технологический стек

| Слой | Технология |
|------|-----------|
| **Frontend** | Angular 21, TypeScript 5.9 |
| **Роутинг** | Angular Router |
| **Reactive** | RxJS 7.8 |
| **UI** | ngx-toastr (уведомления) |
| **Карты** | Yandex Maps API |
| **Тесты** | Vitest 4.0, jsdom |

## API

**Базовый URL:** `https://testologia.ru/delivery/`

| Метод | Endpoint | Описание |
|-------|----------|---------|
| `POST` | `/create` | Создать новый заказ доставки |
| `GET` | `/info?id={id}` | Получить информацию о доставке |

## Установка и запуск

### Требования
- Node.js 18+
- npm 10.9.2+

### Установка зависимостей
```bash
npm install
```

### Запуск разработки
```bash
npm start
```
Приложение доступно на `http://localhost:4200`

### Сборка для production
```bash
npm build
```

### Режим watch
```bash
npm watch
```

### Тесты
```bash
npm test
```
